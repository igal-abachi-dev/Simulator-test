# Simulator v8 Technical Report Draft

## Abstract

Simulator v8 is an observable anomaly sandbox connecting early-universe inflation, autocatalytic chemistry, quantum measurement, black-hole information, and dark-energy sensitivity.

## What This Project Is Not

It is not a proof of the simulation hypothesis, not a claim that the universe contains software bugs, and not a replacement for experimental likelihoods.

## Architecture

The project uses a C# experiment engine for reproducible runs and optional Python scripts for plots and ML surrogate modeling.

## Observable Outputs

- Inflation: scalar power peak, SIGW peak frequency/amplitude, loop-closure error.
- Life: maxRAF size, growth exponent, ignition probability, structural/kinetic mismatch.
- Measurement: CSL collapse rate, visibility time, X-ray proxy, non-standard Born-bias null score.
- Black hole: Page time, final unitarity residual.
- Cosmology: distance residuals and sensitivity proxies.
- Observer: Born-bias power analysis and Wigner-friend-style toy correlations.

## Cross-Domain Scoring

The analysis layer computes cross-module Pearson correlations across shared sweep cases. The default report highlights inflation peak power versus RAF growth exponent as an exploratory pattern, not a causal claim.
