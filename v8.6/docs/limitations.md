# Limitations

1. This project does not prove the simulation hypothesis.
2. Cross-module correlations are exploratory and not causal evidence.
3. The inflation SIGW module uses a proxy, not a full second-order cosmological perturbation solver.
4. The CSL/X-ray module is a scaling map, not an official experimental likelihood.
5. The dark-energy module starts with reference/synthetic curves and should not be presented as a full DESI likelihood.
6. Observer-dependent tests are non-standard-bias/null tests, not a model of consciousness as a physical force.
