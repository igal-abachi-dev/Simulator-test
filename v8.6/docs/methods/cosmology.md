# Cosmology and dark-energy module

The cosmology module is a sensitivity toy model, not a real DESI/Planck likelihood engine.

It computes expansion-history and distance-residual proxies for LCDM, wCDM, and CPL-like `w0-wa` variations. It also records fine-tuning/sensitivity outputs so the analysis layer can identify parameter regions where small changes create large observable residuals.

Real-data likelihood loading is intentionally deferred.
