# Observer-dependent measurement module

This module does **not** model consciousness as a physical force.

It contains three testable toy outputs:

1. `observer_nonstandard_bias_null.csv` — Monte Carlo null test for a non-standard Born-rule probability shift.
2. `observer_born_bias_power_analysis.csv` — sensitivity table showing which probability shifts are detectable at a chosen sigma threshold.
3. `observer_wigner_friend_correlations.csv` — Wigner-friend-style CHSH-like toy correlation table.
4. `observer_contextuality_toy.csv` — observer-context probability-shift table.

Interpretation: a null result is not a failure. It means no non-standard effect was detected above the configured sensitivity threshold.
